# source-logs
Repository to record assessments, tasks and pending issues for data sources loaded to Artsdata. Each source should have a single source-log issue for all graphs loaded to Artsdata (no matter which pipeline). Sub-issues may be created to delegate tasks associated with a source log.

For more information on the steps required to assess, load and auto-mint a source, see this Google doc: [Auto-minting Readiness Protocol](https://docs.google.com/document/d/1OV3EUWl7CE5jauyvvDfv-04yG1fvGADRdd4SaeMOwLg/edit?usp=sharing).
